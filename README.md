## Guía de commits

```
♻️ [Refactor code]
⚡️ [Improve code]
🐛 [Fix bug]
🩹 [Fix non critical bug]
🚑️ [Critical Hotfix]
✅ [Add feature]
✨ [Add small feature]
🚧 [feature not completed]
💥 [Add breaking changes]
➕ [Add dependency]
➖ [Remove dependency]
🗑️ [Remove]
📝 [Document]
🔀 [Merge]
```
